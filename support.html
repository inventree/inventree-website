Get Support

As an open source project we offer a public issue tracker and public documentation.

General Questions

The core team answers questions in the issue tracker of in discussions when they find time.
Private questions (via DM, email, etc.) are generally not answered, use the public forums.

Custom Features

If you require a custom feature you can either:

  • Suggest the feature on the issue tracker as a Feature Request
  • Implement the feature yourself, and contribute back to the project. InvenTree developers will happily help you work through any issues.
  • Create a plugin
  • Request a paid customisation

Paid Customisation / Support

If you use InvenTree for your business and require a support contract or want to fast track a feature request we have a list of developers and businesses willing to help out. Feel free to reach out to support <AT> inventree <DOT> ORG - we will redirect you to a suitable contact.
